# AI ECG Generator - 设计文档

## 项目概述

基于AI的12导联心电图智能生成系统。用户输入生理病理状态描述（或从25种内置模板中选择），系统通过AI模型生成符合临床医学规范的心电参数JSON，并在Canvas上渲染出标准12导联心电图报告。内置模板也可直接渲染，无需AI调用。

**定位**：医学教学与研究工具，非临床诊断设备。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **AI智能生成** | 配置OpenAI兼容API后，输入疾病描述，AI输出结构化ECG参数并渲染 |
| **模板即时渲染** | 25种预定义模板，覆盖正常、心律失常、传导异常、缺血、电解质紊乱等 |
| **12导联报告** | 标准导联布局(I/II/III/aVR/aVL/aVF/V1-V6) + 底部节律条 |
| **高级手动调参** | 23个可配置参数，实时调整波形形态 |
| **导出** | Canvas渲染结果导出为PNG图片 |
| **本地持久化** | AI配置(localStorage)保存，刷新不丢失 |

## 医学实现细节

### 12导联渲染原理

1. **肢体导联**（I/II/III/aVR/aVL/aVF）：通过hexaxial参考系将心电向量（QRS轴/P轴/T轴）投影到各导联角度
   - `amplitude_lead = cos(axis - lead_angle) × base_amplitude`
   - aVR导联自动翻转

2. **胸导联**（V1-V6）：使用R波递增/S波递减模型
   - V1 → rS型（小R深S），V6 → qR型（高R无S）
   - 缺血改变按指定导联范围渲染（如V1-V4 ST抬高）

3. **节律条**：II导联10秒连续波形

### 波形生成

每个心搏由P波、QRS复合波、ST段、T波四部分组成，各用标准波形函数生成（高斯函数逼近P波、多项式逼近QRS、指数函数逼近T波），支持多种病理形态变体（高尖T、倒置T、穹窿型ST等）。

### 高级可调参数

心率、PR间期、QRS时限、QT间期、P/QRS/T波振幅、QRS/P/T电轴、ST抬高/压低、ST段形态、T波形态、T波倒置/高尖/低平、病理性Q波、U波、QRS增宽、RBBB形态、起搏信号、R波切迹、继发性ST-T改变。

## 使用方式

1. **直接模板渲染**：点击快速选择芯片 → 即时显示12导联报告
2. **AI生成**：展开AI配置 → 填入Endpoint/Token/Model → 输入描述 → 点击生成
3. **手动调参**：展开高级选项 → 修改参数 → 点击"应用参数"
4. **导出**：点击工具栏"导出"按钮下载PNG

## 技术栈

- 纯前端，无框架依赖
- Canvas 2D API 渲染
- OpenAI 兼容 API（支持 GPT-4o / DeepSeek / Claude / 本地模型等）
- localStorage 配置持久化
- CSS Grid 全视口布局
